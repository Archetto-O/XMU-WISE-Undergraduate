# XMU-WISE-Undergraduate

XMU 本科WISE班 部分课程的自整理学习资料。

内容包括课程笔记、期中期末复习资料、代码示例及部分通知性文件等，后续小概率会补充和整理。

大部分资料为本人自行编写。小部分资料来自历届学长姐代代相传，有些本人重新整理排版过。内容仅供非商业使用。

没有进行宣传，见到即是有缘。如果有帮助，点个star吧~

## 使用说明

本仓库中的资料几乎必然存在与当前课程要求不完全一致的情况。使用前请结合任课教师的最新教学安排、课程大纲和教材内容自行核对。

代码文件仅作为示例，不保证适用于所有运行环境或课程作业要求。抄袭是极其愚蠢的行为。
